# Lab TryHackME - SQL Injonction Basique

## Objectif
Apprendre à exploiter une vulnérabilité SQLI simple 

## Étapes
1. Identifier le champ vulnérable
2. Injecter une requête simple
3. Observer le comportement 
4. Noter ce que tu apprends

## Résultats
- Injection SQL réussi sur le champ de login

## Leçons apprises
- Importance de la validation côté serveur
- Compréhension basique de SQLi
