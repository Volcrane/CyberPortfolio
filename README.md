# CyberPortfolio

Ce repo contient mes projets pratiques et labs réalisés pour apprendre et appliquer mes compétences techniques

## À propos de moi
Je m'appelle **Jaafar Mohamed**, passionné par la cybersécurité et l’exploration des vulnérabilités web et réseau

## Projets / Writeups

### XSS PortSwigger – Alert(1)
- Compréhension et exploitation d’une vulnérabilité XSS dans un blog
- Analyse des cookies et des requêtes HTTP
- Leçons apprises : importance de la validation côté serveur et encodage des entrées utilisateur

### TryHackMe – Intro to Web Hacking
- Apprentissage des bases de la sécurité web
- Identification des failles simples sur des formulaires web
- Leçons apprises : validation côté serveur, analyse des messages d’erreur, bases de l’exploitation web

## Compétences mises en pratique
