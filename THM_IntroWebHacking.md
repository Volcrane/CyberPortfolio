# Lab TryHackMe – Intro to Web Hacking

## Objectif
Apprendre les bases de la sécurité web et la détection de vulnérabilités simples

## Étapes
1. Explorer la page web et identifier les formulaires
2. Tester des entrées simples pour comprendre la validation côté client et serveur
3. Observer les messages d’erreur pour détecter des failles potentielles 
4. Prendre note des techniques utilisées pour chaque test 

## Résultat
- Compréhension claire du fonctionnement des formulaires web
- Identification des points faibles de validation côté client et serveur

## Leçons apprises
- Importance de la validation côté serveur
- Bases de l’exploitation des formulaires web
